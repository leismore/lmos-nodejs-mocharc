# lmos-nodejs-mocharc

The `.mocharc.json` file for [LMOS for Node.js](https://nodejs.lmos.leismore.org) project.

## Environment

* ts-node 10

## Usage

1. Download `mocharc.json` file
2. Place it at the root directory of your project
3. Rename the file to `.mocharc.json`
